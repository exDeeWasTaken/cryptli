# Cryptli

## Building 
### On Windows
* open powershell
* in powershell: `cd path/to/this/repo`
* in powershell: `./build.ps1`
### On Linux
* open terminal
* in terminal: `cd path/to/this/repo`
* in terminal: `./build.sh`

## Testing
### On Windows
* open powershell
* in powershell: `cd path/to/this/repo`
* in powershell: `./test.ps1`

### On Linux
* open terminal
* in terminal: `cd path/to/this/repo`
* in terminal: `./test.sh`

## Push to git
### On Windows
* open powershell
* in powershell: `cd path/to/this/repo`
* in powershell: `./git.ps1`

### On Linux
* open terminal
* in terminal: `cd path/to/this/repo`
* in terminal: `./git.sh "<commit-message>"`
